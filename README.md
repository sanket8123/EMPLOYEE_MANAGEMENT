# Employee Management System in C

Console-based employee management system written in C.

## Features

- Add employee
- List all employees
- Search employee by ID
- Update employee
- Delete employee
- Save records in `employees.dat`

## Compile

Using GCC:

```sh
gcc employee_management.c -o employee_management
```

On Windows, the output may be:

```sh
gcc employee_management.c -o employee_management.exe
```

## Run

```sh
./employee_management
```

On Windows PowerShell:

```powershell
.\employee_management.exe
```
